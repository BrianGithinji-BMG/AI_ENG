# Brian’s Stock Predictions

**Brian’s Stock Predictions** is a web application that allows users to enter stock tickers (e.g., `AAPL`, `TSLA`) and automatically generates human-readable insights on whether to **BUY**, **HOLD**, or **SELL** each stock.  
The analysis is powered by **Google’s Gemini AI** models.

---

## Overview

This project integrates **Google Generative AI (Gemini)** with a **Vite-based frontend**.  
It simulates stock data and uses an AI prompt to generate brief financial insights.  
The goal is to demonstrate how AI can be applied in a financial web application context.

---

## Features

- AI-generated financial reports using Gemini API  
- Dynamic stock ticker input form  
- Clean and responsive user interface  
- Fast and lightweight performance with Vite  
- Secure handling of API keys using `.env` file
